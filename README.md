# Catálogo de Mangás

Projeto simples desenvolvido para a disciplina de Controle de Versão de Código.

O sistema permite cadastrar mangás com nome e volume, listar os itens e remover da coleção.

## Funcionalidades

- Adicionar mangá
- Listar mangás
- Remover mangá

## Tecnologias

- HTML
- JavaScript

## Versionamento

O projeto segue o padrão de versionamento semântico (MAJOR.MINOR.PATCH).
